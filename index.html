My Projects.

Landing Page Hosting for "Happy Pet" E-commerce Brand on AWS.

Designed and deployed a fully responsive landing page for an e-commerce brand, Happy Pet, using AWS services. Hosted the static website on Amazon S3 with static website hosting enabled, and integrated it with CloudFront to deliver content securely with low latency and HTTPS encryption. The project showcases cloud deployment of production-ready static content without the use of a custom domain.

Full-Stack App Deployment with Terraform on AWS

Provisioned and deployed a full-stack web application on AWS using Terraform. The infrastructure includes a React frontend hosted on S3, a Node.js backend on EC2, and a PostgreSQL database on RDS. Entire infrastructure is automated using Infrastructure as Code principles.
